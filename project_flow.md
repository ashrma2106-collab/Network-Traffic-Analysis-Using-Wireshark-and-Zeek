# Project Workflow

1. Capture network packets using Wireshark.
2. Save captured packets as a `.pcap` file.
3. Analyze packets using Wireshark filters.
4. Generate network logs using Zeek.
5. Review suspicious traffic patterns.
6. Document findings and recommendations.
