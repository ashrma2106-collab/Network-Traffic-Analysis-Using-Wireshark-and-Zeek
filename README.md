# Network Traffic Analysis Using Wireshark and Zeek

## 📖 Overview

This project demonstrates how network traffic can be captured, monitored, and analyzed using Wireshark and Zeek. It focuses on understanding network communication, detecting suspicious activities, and improving cybersecurity through packet inspection and log analysis.

---

## 🎯 Objectives

- Capture live network traffic.
- Analyze different network protocols.
- Detect suspicious or malicious traffic.
- Generate and analyze Zeek logs.
- Improve understanding of network security monitoring.

---

## 🔄 Methodology

### Step 1 – Packet Capture
Capture network packets using Wireshark.

### Step 2 – Traffic Filtering
Apply filters to inspect specific protocols and conversations.

### Step 3 – Protocol Analysis
Analyze TCP, UDP, HTTP, DNS, and ICMP traffic.

### Step 4 – Threat Detection
Identify suspicious packets, unusual connections, and abnormal network behavior.

### Step 5 – Log Generation
Generate detailed network logs using Zeek.

### Step 6 – Reporting
Document observations and provide security recommendations.
